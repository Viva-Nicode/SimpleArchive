# 📦 Simple Archive

**간편한 나만의 로컬 저장소**  
다양한 유형의 데이터를 한 곳에서 쉽고 직관적으로 관리할 수 있는 iOS 애플리케이션입니다.

불필요한 설정 없이,  
폴더와 페이지를 통해 메모·정보·기록을 자유롭게 저장할 수 있습니다.

---

## 🛠 프레임워크

- Swift
- UIKit
- Combine / Swift Concurrency
- URLSession
- SFBAduioEngine
- ZIPFoundation
- Local Data (UserDefaults / CoreData / 기타)
- XCTest
